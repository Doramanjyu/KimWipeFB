# キ○ワイプファンブック(仮)

*キムワイプにキムワイプ擬人化マンガを印刷した同人誌*

## 登場キャラクター

* キムワイプちゃん (日本製紙クレシア)
* Kimwipes-chan (Kimberly-clark)
* めがねおにいさん

## 内容

* 本家と日本の柄の違い
  - 日本のは円のデザイン
  - 本家はグラデーション
* 用途
  - 光学を含む各種拭き取り
  - 弱い表面コーティングがあると傷をつける可能性
* 都市伝説
  - ガラス繊維が入っているのでガラスを傷つける
  - んなわけない
  - 埃等をきちんと落とさずに拭いたり、パルプ以下の硬度のコーティングをこすれば、もちろん傷かつく
